Detailed Description

The Organism class represents a living thing.

The main purpose of the Organism is to store categorical information, such as scientific name, as well as the genotype. New organisms can be created through use of the LoveChamber class and two parent organisms.

See also
Gene, LoveChamber

Constructs an Organism with an initial genus, species, name, and blank genotype.

As the initial state of the Organism is one without a genotype, it is necessary that ::addGene is used to assign the Organism some genotype which then makes it elligble to mate with other organisms.
